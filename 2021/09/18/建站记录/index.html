请结合hexo官方文档观看

本地操作
安装 git Node.js

使用npm安装hexo 实际是hexo-cli

初始化
$ hexo init folder-name
$ cd folder-name
$ npm install

folder-name 改成自己的文件夹名称

$ hexo s
从本地预览，初始是一个Hello World!以及官方自带的主题

github操作
github上新建仓库
要求 ： 必须以 username.github.io 命名
其中 username 必须是github账户名

本地操作
在前面的 $ hexo init 命令所创建的文件夹中找到_config.yml，修改deploy下的选项

注意：冒号后面必须有一个空格，repo还有其他写法，但不再介绍，后两个可选

之后
$ npm install hexo-deploy-git –save
$ hexo clean && hexo deploy

github操作
前往github,到所建的仓库下，在仓库的设置中把默认分支改成master,因为_config.yml里的默认分支就是master。愿意折腾的可以自己改。

其他
如果想要更改主题，请到hexo官网找到主题下载（其实是从github上下载），下载到theme本地文件夹。之后去_config.yml里改对应设置。最好执行一下 $ hexo d

注意：本地更改过后有可能无法在网页上显示改变，可能需要等待一会。如果长时间没有反应，请执行$ hexo clean && hexo generate && hexo deploy ，如果依然没有反应，这边建议直接在本地重新开一个。

更多内容请参考官方文档
