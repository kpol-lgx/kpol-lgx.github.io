建站记录

2021-09-18
#note

安装 git Node.js
使用npm安装hexo 实际是hexo-cli

初始化
$ hexo init
$ cd
$ npm install

$ hexo s
从本地预览

github上新建仓库
要求 ： 必须以 .github.io 命名
其中 是github账户名

在前面的 $ hexo init 命令所创建的文件夹中找到_config.yml，修改deploy下的选项

  deploy:
  type: git
  repo: <repository url> #https://bitbucket.org/JohnSmith/johnsmith.bitbucket.io
  branch: [branch]
  message: [message]```

之后
$ npm install hexo-deploy-git --save

$ hexo clean && hexo deploy

前往github,到所建的仓库下，在仓库的设置中把默认分支改成master,因为_config.yml里的默认分支就是master.

如果想要更改主题，请到hexo官网找到主题下载（其实是从github上下载），下载到theme本地文件夹。之后去_config.yml里该对应设置

注意：本地更改过后有可能无法在网页上显示改变，可能需要等待一会。如果长时间没有反应，请执行$ hexo clean && hexo generate && hexo deploy ，如果依然没有反应，这边建议直接在本地重新开一个。
另外，由于github是境外网站，网速可能很慢，所以执行命令就慢慢等好了，超时就重新执行
